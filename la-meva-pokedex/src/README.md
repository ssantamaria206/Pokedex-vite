# La Pokédex Reactiva

**Enunciat:**

Crea una funció `cercarIComparar(nom1, nom2)` que:

1. Busqui els dos Pokémon en **paral·lel** (`Promise.all`).
    
2. Gestioni els **errors** amb un únic `try/catch`.
    
3. **Mapegi** les dades per quedar-se només amb el `name` i el `base_experience`.
    
4. Imprimeixi per pantalla quin dels dos té més experiència base.
